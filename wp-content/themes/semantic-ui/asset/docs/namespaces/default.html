template_part__locateÂ¶

A drop-in replacement for WordPress' locate_template()

NOTE: This is almost entirely copied from the WordPress 4.2.2 core

Improvements:

  • Function variables do not interfere with included file
  • Uses is_file() instead of file_exists()
  • Better Debugging
package

Default
see https://codex.wordpress.org/Function_Reference/locate_template

Arguments

$template_names

string|array

Template file(s) to search for, in order.

$load

boolean

If true the template file will be loaded if it is found.

$require_once

boolean

Whether to require_once or require. Default true. Has no effect if $load is false.

Response

string

The template filename if one is located.
